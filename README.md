#Frontend

Aplikacja została zaimplementowana w stacku: Typescript + React + Vite

Zgodnie z wymaganiami aplikacja posiada:

- Trzy wykresy kołowe przedstawiające udział różnych źródeł energii wraz z średnim procentowym udziałem czystych źródeł energii na dane dni
- Możliwość uzyskania informacji na temat najlepszego okna czasowego do ładowania w dwóch prognozowanych dniach
